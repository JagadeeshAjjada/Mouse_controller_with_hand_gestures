Simple Mouse-Controller with hand gestures using pynput.

### Setup
Create project with virtual environment

```console
$ mkdir myproject
$ cd myproject
$ python -m venv venv
```

Activate it on Linx
```console
$ . venv/bin/activate
```

or on Windows
```console
$ venv\Scripts\activate
```

Install required packages
```console
$ pip install -r requirements.txt
```

Run the app
```console
$ python main.py
```
